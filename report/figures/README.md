# 報告書用画像の配置

`main.tex` から参照している画像ファイルを、このディレクトリに配置してください。

## 推奨ファイル名

| ファイル名 | 内容 | 推奨 |
|------------|------|------|
| `sfmmap_overview.png` | SFMMap 全体俯瞰（2F・1F が分かる） | PNG または PDF |
| `sfmmap_2f.png` | 2階平面図（部屋・階段出口_2F・経路網） | 同上 |
| `sfmmap_1f.png` | 1階平面図（部屋・階段入口_1F・階段） | 同上 |
| `sfmmap_layers.png` | レイヤー一覧または領域の色分け表示 | 同上 |
| `simulation_log.png` | シミュレーション画面＋ログ（任意） | 同上 |

## エクスポートのヒント

1. S-quattro の地図エディタ（SFMMap）で対象階を表示
2. レイヤーの表示／非表示で必要な領域だけ見せる
3. スクリーンショットまたは「画像として保存」で PNG を出力
4. 解像度は横幅 1200〜2000 px 程度を推奨

画像が無い状態でも `main.tex` はプレースホルダ枠付きでコンパイルできます。
